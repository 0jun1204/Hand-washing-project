# 뽀-득 (ppo-deuk)

## 📋 프로젝트 소개

**뽀-득**은 웹캠을 활용한 실시간 손씻기 6단계 검사 AI 프로그램입니다. YOLOv8 모델을 기반으로 올바른 손씻기 동작을 인식하고 가이드하는 디지털헬스케어 솔루션입니다.

### 🏆 수상 이력

- **지자체 연계 디지털헬스케어 교육 과정 경진대회 대상(경상남도지사상)** - AI 디지털헬스케어부
- **경남대학교하계 일머리 사관학교 대상(경남대학교총장상)** - AI 디지털헬스케어부

## ✨ 주요 기능

- **실시간 손씻기 동작 인식**: 웹캠을 통해 손씻기 동작을 실시간으로 감지
- **6단계 순차 검사**: WHO 권장 손씻기 6단계를 순차적으로 검사
- **시각적 가이드**: 각 단계별 이미지와 진행률 표시
- **음성 안내**: 단계별 음성 가이드 제공
- **실시간 피드백**: 진행 상황을 시각적으로 확인 가능

## 🧼 손씻기 6단계

1. 손바닥을 마주 대고 씻어주세요
2. 손등과 손바닥을 마주 대고 씻어주세요
3. 손깍지를 끼고 씻어주세요
4. 손가락을 마주 잡고 씻어주세요
5. 엄지손가락을 돌려주며 씻어주세요
6. 손톱 손바닥에 문지르며 씻어주세요

## 🛠️ 기술 스택

- **Python 3.8**
- **PySide6**: GUI 프레임워크
- **OpenCV**: 웹캠 영상 처리
- **PyTorch**: 딥러닝 프레임워크
- **Ultralytics YOLOv8**: 객체 감지 모델
- **pygame**: 음성 재생

## 📦 설치 방법

### 필수 요구사항

- Python 3.8 이상
- 웹캠

pip install torch torchvision

## 🚀 사용 방법

1. 프로젝트 디렉토리로 이동
2. 필요한 모델 파일(`_internal/best.pt`)이 있는지 확인
3. 프로그램 실행:
```bash
python main.py
```

### 키보드 단축키

- **F11**: 전체화면 모드 전환
- **ESC**: 전체화면 모드 해제

## 🤖 AI 모델

- **모델**: YOLOv8 (Ultralytics)
- **학습 데이터**: 직접 생성 및 라벨링된 손씻기 5단계 데이터셋
- **모델 파일**: `_internal/best.pt`

### 모델 사용법

모델 파일은 코드에서 자동으로 로드됩니다:
```python
self.model = YOLO('_internal/best.pt')
self.model.classes = [0, 1, 2, 3, 4, 5]  # 6가지 손씻기 동작 클래스
```

## 📊 작동 원리

1. **초기 감지**: 웹캠에서 손이 감지되면 프로그램 시작
2. **순차 검사**: 6단계 손씻기 동작을 순서대로 검사
3. **진행률 추적**: 각 단계마다 최소 15회 감지가 필요
4. **시각적 피드백**: 
   - 진행률 표시 (빨간색 → 노란색 → 파란색)
   - 단계별 시그널 표시
   - 실시간 카메라 화면
5. **완료 처리**: 모든 단계 완료 시 완료 메시지 표시
